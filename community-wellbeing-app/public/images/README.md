# Images Directory

This directory contains all the images used in the Community Wellbeing App.

## Structure

- **events/** - Event images (yoga-park.jpg, pottery-workshop.jpg, trail-run.jpg, meditation.jpg)
- **communities/** - Community banner images (outdoor-enthusiasts.jpg, wellness-warriors.jpg)
- **organizations/** - Organization logos (wellness-hub-logo.png, creative-minds-logo.png)
- **avatars/** - User avatar images (default-avatar.png)

## Image Guidelines

For a calming, wellbeing-focused aesthetic:

- Use soft, natural colors (sage greens, calm blues, warm earth tones)
- Prefer images with natural lighting
- Include nature, peaceful settings, and joyful people
- Maintain transparency and lightness in compositions
- Recommended dimensions:
  - Event images: 800x600px
  - Community banners: 1200x400px
  - Organization logos: 400x400px (transparent background)
  - Avatars: 200x200px (circular crop)
